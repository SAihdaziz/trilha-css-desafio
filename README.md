## 📌 Sobre o projeto
Este projeto consiste na estilização de uma landing page utilizando **CSS puro**, a partir de uma estrutura HTML previamente fornecida.

O foco foi aplicar conceitos de layout, design e responsividade para transformar uma base simples em uma interface moderna e profissional.

## 🎨 Foco do desenvolvimento
- Estilização completa da interface com CSS  
- Aplicação de Flexbox para alinhamento e layout  
- Uso de background com imagens e overlay  
- Criação de efeitos visuais (hover e parallax simples)  
- Organização e estruturação do CSS
